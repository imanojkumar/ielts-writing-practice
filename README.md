# IELTS Writing Practice Dashboard

**Free, real-exam simulation tool for IELTS Writing (Task 1 + Task 2)**

- Choose **General Training** or **Academic**
- Full Mock / Task 1 only / Task 2 only
- Built-in 20-min + 40-min timers
- Live word counters
- Cambridge-style prompts (Books 10–21 patterns)
- One-click “Copy Full Submission for Grok” for detailed band feedback

## How to Use

1. Open the live site
2. Choose **General Training** or **Academic**
3. Select a prompt from the dropdown
4. Write under timed conditions
5. Click **Copy Full Submission for Grok**
6. Paste into Grok for full 4-criteria feedback + Band 8–9 models

## Features

- Dark mode, clean UI
- No login, no ads, no tracking
- Works completely offline once loaded
- Keyboard shortcut: `Ctrl + Enter` to copy submission

## Local Development

Just open `index.html` in any browser. No build step required.
